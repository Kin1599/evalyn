ℹ️ Информация

Evalyn — это платформа для помощи преподавателям в проверке лабораторных работ.

Возможности:

  • ✨ Автоматическая проверка кода с использованием AI
  • 🔍 Детальный анализ ошибок и рекомендации
  • 📊 Просмотр результатов проверки и статистики
  • 🐍 Безопасное выполнение Python кода в изолированной среде
  • 📚 Поддержка Jupyter ноутбуков
  • 💬 Отправка фидбека студентам через Telegram

Начните работу: Используйте кнопки выше для просмотра ваших курсов и заданий.
