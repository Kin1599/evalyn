    │       • shutdown — корректное завершение
    │
    └── [ ] tests/test_kernel_orchestrator.py (с моками ipykernel)

               пулом ядер и обрабатывает все команды клиента.

 Как использовать этот план

═══════════════════════════════════════════════════════════════════════════════

1. Прочитать IMPLEMENTATION_PLAN.md для деталей каждого этапа
2. Начать с Этапа 2: Провайдеры (Этап 1 уже готов)
3. После каждого этапа:
   - Запустить тесты: pytest tests/test_<component>.py -v
   - Обновить документацию
   - Создать примеры использования
   - Выполнить code review
4. При завершении Этапа 5 — полная готовая библиотека
