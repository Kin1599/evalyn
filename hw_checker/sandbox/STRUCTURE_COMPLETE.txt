   - tests/test_venv_storage.py
   - tests/test_gvisor_provider.py

3. Для каждого компонента:
   ├─ Определить интерфейс (в переменных)
   ├─ Написать тесты (TDD подход)
   ├─ Реализовать компонент
   ├─ Запустить: pytest tests/test_<component>.py -v
   └─ Обновить документацию

4. После Этапа 2 перейти к Этапу 3, и так далее

5. Проверьте:
   • pytest tests/ -v           # Все тесты
   • mypy sandbox/              # Типизация
   • black sandbox/              # Форматирование
   • ruff check sandbox/         # Качество кода
